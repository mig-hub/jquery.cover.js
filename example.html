Options

OptionDefault
bindingclick.coverThe event that should trigger the fullscreen.
divIDcover-plugin-blocThe ID property of the div created by the plugin.
loaderloader.gifThe address of an image displayed while the fullscreen image is loading. Most likely an animated gif. The default uses a loader that ships with the plugin.
backgroundColortransparentBackground color of the div when it is loading the fullscreen image.
preloadtrueBoolean. If this is true, the plugin will try to preload the fullscreen images as long as they are set when the plugin is called.
duration200The duration of the scaling animation.
